# Kasaz(Test Project) Ruby on Rails + GraphQL Backend (API only)

* Main tech stacks

     Ruby on Rails, GraphQL, PostgreSQL, RSpec

* Versions

     ruby '2.6.6'

     rails '~> 6.0.3', '>= 6.0.3.2'

* Setup environment

     `bundle install`

* Running backend on local

     `rails s`

* Generate random data

     `rails db:seed`

* Running unit test

     `rails spec`
